## Prompt

```text
/sp.plan # Implementation Plan: Fix Build Failure (Sidebar ID Mismatch)

**Error**: Build fails because `sidebars.ts` uses numbered IDs (`01-...`) but Docusaurus strips numbers during build.
**Solution**: Update `sidebars.ts` to use `autogenerated` slices pointing to directory names. This delegates ID resolution to Docusaurus while keeping our Manual Categories (Arrows).

## Phase 1: Fix Sidebar Configuration
1. **Overwrite `sidebars.ts`**:
   - **Action**: Replace the entire file with this robust configuration.
   - **Strategy**: Define the Label manually (for the Arrow), but auto-generate the content from the folder.
